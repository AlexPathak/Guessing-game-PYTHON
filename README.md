Number Guessing Game

This is a simple Python game I built while learning programming. In this game, the computer randomly selects a number between 1 and 100, and the player has to guess it within a limited number of attempts. After each guess, the game gives hints like whether the correct number is higher or lower, which helps the player narrow it down.

While building this project, I practiced basic Python concepts such as loops, conditional statements, user input, and the random module. It really helped me understand how logic is built step by step in programming.

I made this as a beginner practice project to strengthen my Python basics and improve my problem-solving skills. It may be simple, but it represents one of my early steps in learning programming 🙂
